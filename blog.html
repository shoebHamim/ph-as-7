1. Difference between local storage and session storage.

-Session storage is temporary, data is stored for the duration of a session.It get expired when the browser or tab is closed.
Local Storage is somewhat permanent.It must be deleted manually. It doesn't expire on browser/page reload/restart.

- Session storage usually stores small amount of data while local storage often stores larger amount of data.

2. Global scope Vs Block Scope.

- Global scope means the entire coding scope. Variable that is declared in global can be accessed from anywhere of the code.
On the other hand, local scope is small portion of that code like variable declared inside a function. That variable can only be accessed inside that particular function.

3. How does event loop in js work?

- JS event loop monitors the Call Stack and the Call back Queue.If the Call Stack is empty, the event loop takes the first event from the queue
and pushes it to the call stack where the event gets executed. Here Web apis helps event loop to handle asynchronous events. This is how event loop in JS works.

4. What are the ways to get undefined in JS?

- Printing a variable that has not been assigned to a value.

- Function that has no return statement gives undefined.

- Trying to access non-existing property of an object.

- Trying to access out of index value in array.
